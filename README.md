# bspwm

A collection of my dotfiles for my Nord based bspwm setup. Running on Arch.
Folder structure assumes all folders in the .config folder in the Home directory.

Apps used:

bspwm (window manager)<br>
dunst (notifications)<br>
rofi (application launcher)<br>
sxhkd (keyboard shortcuts)<br>
flameshot (snipping tool)<br>
kitty (terminal emulator)<br>
picom (compositor)<br>
htop (resource management)<br>
